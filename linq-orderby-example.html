LINQ OrderBy Example in C#

OrderBy is used in LINQ to sort elements in ascending order based on a key. It is commonly used when working with collections in C#.

Basic Syntax

var sorted = data.OrderBy(x => x.Age);

OrderByDescending Example

var sorted = data.OrderByDescending(x => x.Age);

Practical Example

var students = data
    .Where(x => x.Age > 20)
    .OrderBy(x => x.Name)
    .Select(x => new { x.Name, x.Age });
